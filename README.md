# Blockchain Galaxy

Phase 1 foundation for a Vite + React + TypeScript 3D website.

## Commands

```bash
pnpm install
pnpm dev
pnpm build
pnpm preview
```

## Deploy to Vercel

```bash
pnpm dlx vercel login
pnpm dlx vercel
pnpm dlx vercel --prod
```

Use `pnpm build` as the build command and `dist` as the output directory.
